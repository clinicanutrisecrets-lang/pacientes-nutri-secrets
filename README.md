# 🌿 Nutri Secrets

Aplicativo de nutrição terapêutica personalizada com base científica.

## Como publicar no Vercel

### Opção 1 — Via GitHub (recomendado)
1. Crie um repositório no GitHub
2. Faça upload desta pasta
3. Acesse vercel.com → "New Project" → importe o repositório
4. Clique em Deploy ✅

### Opção 2 — Via Vercel CLI
```bash
npm install -g vercel
cd nutri-secrets
vercel
```

## ⚠️ Importante: Chave da API Anthropic
O app usa a API da Anthropic. Você precisa configurar sua chave.
Veja as instruções em SETUP.md.
